# ElderConnect

A prototype community support platform for elderly users.

## Setup

1. Copy `.env.example` to `.env` and set `EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY`.
2. Start the app: `bun run start` (Expo)

## Usage

- **Sign up** / **Sign in** with email and password
- **Events** – Browse and register for community events
- **Caretakers** – View caretakers and book (Day / Week / Month / Custom)
- **Profile** – View account and sign out

## Tech

- Expo Router, React Native
- Clerk (email+password)
- SQLite (local)
- Bun
